#### 使用方法

##### 克隆仓库
`git clone https://github.com/EnderRomantice/Prime_Key-create-space`

##### 安装Node.js依赖
`cd frontend`
`pnpm install`

#### 技术选型

##### 前端

基本组件: React + TSX, 部分特效来自reactbits, 所有CSS样式使用Tailwind CSS

Markdown渲染 + 代码高亮: react-markdown, prism-react-renderer

网络请求: fetch

##### 后端

Python + FastAPI + SQLite
